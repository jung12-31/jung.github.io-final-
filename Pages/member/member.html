團隊介紹

「微熱邱林」背後的推手，用專業與熱情為您烘焙每一份幸福

邱苡容

全站頁面連動、設計統整與技術整合(負責跨頁面功能銜接、視覺與維持互動一致性，並進行最終統整與測試)

在本次網站專案中，我主要負責商品評論系統、會員中心的訂單與評論紀錄、結帳頁面，以及整體前端功能的整合與連動。專案全程使用前端技術完成，透過 JavaScript 控制頁面邏輯與互動，並以 LocalStorage 模擬資料儲存與狀態管理，未實際串接後端。 由於專案無法即時共編，我身為文件持有人，需負責整合各組員完成的功能。整合過程中，因程式碼註解不足、修改未標示等問題，常需逐行比對與修正，花費大量時間排除衝突與錯誤，部分功能甚至必須重新撰寫。雖然過程中多次感到挫折，但當功能成功串接運作時，也帶來相當大的成就感。 在會員系統方面，我重新調整登入流程，透過 LocalStorage 儲存會員狀態，使登入後能正確顯示會員資訊與紀錄。結帳頁面則由我自行規劃與實作，需處理登入限制、訂單產生、訂單紀錄顯示與頁面連動等邏輯，在未使用後端的情況下，需反覆測試與調整資料流程。 商品評論系統是整體專案中最具挑戰性的部分，需同時處理會員狀態、星等即時更新、評論顯示與會員中心同步紀錄，使前端邏輯相當複雜。過程中也多次遇到顯示與初始化錯誤，只能透過不斷除錯與重構解決。 透過這次團體專案，我深刻體會到前端整合、狀態管理與溝通協調的重要性。雖然整合壓力大、修改次數多，但也讓我在跨頁面資料連動與除錯能力上獲得很大的成長，成為一次相當寶貴的學習經驗。

邱婧貽

包裹進度追蹤、聯絡我們、會員中心(建立物流追蹤訂單、聯絡資訊與帳戶安全功能介面)

在本次期末專題中，我主要負責包裹進度追蹤、會員中心管理及聯絡我們頁面的實作與整合。專題初期，我將設計稿轉化為實際網頁架構，卻發現靜態設計與動態實作之間存在落差，過程中多次感到挫折，也因此體會到網頁開發的難度。 在與組員同步開發的過程中，我們發現網頁製作無法短時間完成，而是需要不斷協調與調整。透過反覆統一色彩、字體與元件樣式，網站逐漸建立起整體一致的視覺風格，也讓我理解到團隊合作在專案中的重要性。 進入功能開發後，最大的挑戰在於跨頁面的資料連動。包裹追蹤與會員登入需共享狀態，讓我意識到前端不只是單一頁面的程式撰寫，而是必須妥善規劃整體結構與資料管理。由於功能由不同組員分工完成，我們必須先建立共用介面，再進行整合與調整，這讓我深刻體會到溝通與系統設計的重要。 透過這次專題，我更清楚認識到自身能力與實務需求之間的差距，也理解到網頁開發結合了視覺設計、邏輯規劃與團隊合作。這次經驗不僅提升了我的前端技術，也讓我在問題拆解與溝通協作上獲得寶貴成長。

林亭羽

商品介紹、期末報告(撰寫商品內容、設計商品展示架構，並統整專案報告)

在這次網頁製作中，我主要負責商品介紹相關頁面，包括商品總覽、商品個別介紹及各分類商品的呈現。一開始以為只是將商品顯示在網頁上，但實際製作後才發現，從資料結構、版面配置到程式碼連動，每個細節都必須事先規劃，否則容易出現跑版或顯示錯誤。 在製作商品總覽與分類時，我最大的困難是結構規劃與檔案整合。由於 HTML、CSS 與 JavaScript 需同時配合，當畫面異常時，往往需要花費許多時間逐一檢查。過程中也因命名錯誤或層級不清，導致樣式無法套用，讓我深刻體會到結構與命名一致的重要性。 整個實作過程中，最具挑戰的是在畫面不如預期時，必須冷靜分析原因，而不是盲目修改。透過反覆嘗試與修正，我逐漸學會以更有系統的方式排除問題。 透過這次經驗，我不僅熟悉了商品分類與總覽頁面的製作，也更理解「結構先行、樣式後調」的概念，並提升了面對錯誤時的耐心與解決問題的能力，為後續的程式開發打下良好基礎。

林若歆

關於我們、門市諮詢、近期活動(負責品牌故事、實體門市資訊呈現與活動頁面規劃)

在本次「微熱邱林」甜點品牌網站專案中，我主要負責「關於我們」、「門市資訊」與「近期活動」頁面的設計與實作。這些頁面是品牌形象的重要呈現，因此在規劃時，我以奶茶色系為主軸，著重排版與視覺細節，營造溫暖、療癒的品牌氛圍。 實際進入開發與整合階段後，才發現即使是以資訊呈現為主的頁面，仍需花費大量時間進行調整與校正。受限於協作環境與檔案連動，我必須反覆檢查各頁面按鈕與路徑，確保功能能正常運作，過程相當考驗耐心。 此外，我也自行研究並製作首頁廣告輪播，在未學過相關技術的情況下，透過查找資料完成版面設計，並反覆調整圖片與呈現效果，使活動與新品資訊能自然融入網站風格。 在整合過程中，跨檔案除錯是最大的挑戰。由於組員撰寫風格不同，需在 HTML 與 JavaScript 之間來回檢視與理解他人邏輯，使除錯難度提升。這也讓我體會到程式結構清楚與一致性的重要。 透過這次專案，我深刻理解到一個看似簡潔的網站，背後其實是大量細節調整與反覆除錯的成果。最終看到網站順利呈現出品牌溫柔、流暢的體驗，讓我覺得所有付出都相當值得，也提升了我在前端實作與問題解決上的能力。
